Fundamentals of User Interface Design

The fundamentals of user interface design are a science. They can be learned by anyone, applied to any user interface, and always improve the ease in which that interface is consumed. How can I make such an broad statement like this? Because we're human, and for the most part we all function in a very similar way.

Grouping

Grouping is a fundamental tool for creating content relationships and designing user interfaces that are are intuitive. When properly used, grouping helps to clearly communicate meaning by giving users visual cues about content relationships and reducing the amount of mental effort used to consume content. Let's take a look at a basic example of grouping letters and see how it drastically affects the meaning of the content.

Unfortunately grouping is often overlooked in the more subtle aspects of user interface design, and content relationships become either weak or misleading. Weak content relationships directly impact usability and cause consumers to expend more mental effort figuring out what is being communicated. Here's a common example — forms.

Ambiguous Grouping

Even spacing causes users to question if the text belongs to the form field above or below it.

Clear Grouping

Varied spacing gives users visual indication that the text is related to the form field below it.
